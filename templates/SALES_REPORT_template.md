## 부록

### A. 관여율 / 해결율 분석 방법론

1. **관여율**: 챗봇이 해당 문의에 초기 응대(분류·정보 수집·안내)를 시작할 수 있는지 여부로 판정. 부분 처리 후 상담원 연결 포함.
2. **해결율**: 모든 처리 단계를 지식(RAG)/태스크(TASK)/규칙으로 자동 완결할 수 있는 케이스 비율. 상담원 개입이 단 한 번이라도 필요하면 미해결 분류.
3. **가중 평균**: 각 SOP의 해결율 × 커버리지 건수로 전체 가중 평균 산출.

### B. 용어 설명

| 용어 | 설명 |
|------|------|
| HT (How-To) | 정보 제공 및 프로세스 안내 유형 SOP |
| TS (Troubleshooting) | 문제 진단 및 해결 유형 SOP |
| **지식(RAG)** | 문서·FAQ 기반 자동 응답 (별도 API 불필요) |
| **태스크(TASK)** | 외부 시스템 API 호출 (주문 조회, 배송 추적 등) |
| **규칙** | 조건 기반 분기 처리 (보증 기간 체크, 에스컬레이션 등) |
| 관여율 | 챗봇이 응대를 시작할 수 있는 문의 비율 |
| 해결율 | 상담원 개입 없이 완전 처리 가능한 문의 비율 |
